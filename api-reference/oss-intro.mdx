### Directory of Open Source Software
<Tip>The Directory of OSS is constantly updating with new primitives for you to explore, check back frequently.</Tip>

<CardGroup cols={1}>
  <Card title="SSM - A single sided orderbook for stake_accounts." icon="github-alt">
    Place bids for Solana (SOL) within stake_accounts at a premium i.e. 0.98 SOL per SOL in stake_accounts. Sell atomically with zero risk.

    <br/>
    <a href="">Read about SSM.</a> 
  </Card>
  <Card title="TRD - Tokenised Revenue Distribution" icon="github-alt">
    Issue a central bank like bond (yield earning) system with Liquid Receipt Tokens (LRTs). Allows for target rates onchain governed by treasurer. 

    <br/>
    <a href="">Read about TRD.</a> 
  </Card>
  <Card title="SRL - Stake(Self) Repaying Loans" icon="github-alt">
    Utilise stake_accounts as collateral in P2P lending markets, allowing for low borrowing costs, high lending rates or even self-repaying loan agreements. 

    <br/>
    <a href="">Read about SRL.</a> 
  </Card>
  <Card title="FRX - Fixed Rate Currency Exchange" icon="github-alt">
    A synthetic pegged currency exchange with USD Reserves that manages the supply and exchange of world currencies pegged to the US Dollar.

    <br/>
    <a href="">Read about FRX.</a> 
  </Card>
  <Card title="IBO - Initial Bond Offering" icon="github-alt">
    Issue onchain bonds for future released tokens that are both tradeable and redeemable without the need for issuing onchain tokens pre-emptively. i.e. Compliant onchain IPO bonds.

    <br/>
    <a href="">Read about IBO.</a> 
  </Card>
</CardGroup>
